# iSee
iSee 是一款资产收集并整理的工具，可以从大量杂乱的文本信息中提取出资产信息(如域名,ip,b段,c段)，从而帮助渗透测试人员收集目标更多的资产       

     
### 用法         

          从指定文本中提取资产:   python3 iSee.py -f text.txt
          指定输出结果保存路径:   python3 iSee.py -f text.txt -o report.txt
